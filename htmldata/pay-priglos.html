Задача этого смарт-контракта
Данный смарт-контракт создается с целью привлечения людей к участнию для деятельности в проекте "Благополучие" и соответствует техническим условиям по безопасности, защищен от изменения своего кода, и отвечает за автоматизацию процесса начислений и выдачи активов, являющихся ему подконтрольными, и им управляемые.
Задача участника
Тщательно изучить предлагаемую информацию, в случае согласия с принятием условий - в точности выполнить инструкции.
Условия для начала работы
Участник предоставляет номер счета в системе, который будет привязан к этому смарт-контракту до окончания условий. Для этого он использует либо один из ранее созданных, либо создает новый, и, с помощью интерфейса управления из раздела "ТВОИ АКТИВЫ" осуществляет привязку этого смарт-контракта. После окончания привязки запустится процесс, который в течении 10 дней принесет тебе 33% профита, на активы, подаренные в этом предложении.
Завершающий этап взаимодействия
По окончании работы смарт-контракта, на счет, привязанный к нему, будет начислена сумма, подаренная в качестве пая, и 33% прибыли на эту сумму. Второй раз этот контракт сработает только с другим счетом, но больше даримой суммы для этого приватного ключа - не будет и покупать пай придется за "свои".
